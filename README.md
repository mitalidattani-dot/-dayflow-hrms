# -dayflow-hrms
## Problem Statement
Manual HR processes such as attendance tracking, leave management, employee records, 
and approval workflows are time-consuming, error-prone, and inefficient. 
Organizations need a centralized digital system to manage HR operations smoothly 
while ensuring transparency for employees and control for HR/Admin

## Solution
Dayflow is a Human Resource Management System (HRMS) designed to digitize and 
streamline core HR operations. The system provides role-based access for Employees 
and Admin/HR officers, enabling efficient employee management, attendance tracking, 
leave approvals, and payroll visibility through a single platform.

## Key Features

### Authentication & Authorization
- Secure Sign Up / Sign In
- Role-based access (Employee / Admin / HR)

### Employee Features
- View and update personal profile
- Check daily and weekly attendance
- Apply for leave (Paid, Sick, Unpaid)
- View leave status (Pending / Approved / Rejected)
- View salary details (read-only)

  ### Admin / HR Features
- Manage employee profiles
- View and manage attendance of all employees
- Approve or reject leave requests
- Manage payroll and salary structure
- Access analytics and reports

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend (planned): Python / Node.js
- Database (planned): MySQL / MongoDB


## User Roles
- *Employee*: Views personal data, attendance, leave, and payroll
- *Admin / HR Officer*: Manages employees, approvals, and HR operations

## How to Run the Project
1. Clone or download the repository
2. Open index.html in a browser
3. Navigate through login and dashboard pages
